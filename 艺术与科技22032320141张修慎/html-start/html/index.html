学校简介

成都东软学院

精勤博学，学以致用

成都东软学院（Chengdu Neusoft University） 是经国家教育部批准设立，由东软出资举办的一所民办普通高等院校，入选四川省卓越工程师教育培养计划，是四川省民办教育协会会长单位。2003年，成都东软信息技术职业学院成立，并被教育部确定为国家级示范性软件职业技术学院；2011年经教育部批准，学校升格为成都东软学院；2015年通过新建本科院校学士学位授予单位评估。 据2023年1月学校官网显示，学校占地面积52.85万平方米，总建筑面积19.85万平方米，共有在校生21000余人；设有12个二级学院和2个教学部，设置32个本科专业,学校共有专任教师772人，其中具有高级职称232人。
